# Итоговый проект курса «Разработка в «не-книжных» условиях»
## Автор: Лимаренко Дмитрий, 2022.

### Архитектура приложения
Трёхуровневая архитектура приложения построена на основе **MERN-стека**. 
Верхний уровень — **React.js**, декларативная среда JavaScript для создания динамических клиентских приложений в HTML. 
Серверная часть построена с использованием **Express.js**, работающим на сервере **Node.js**. 
Облачная база данных — **MongoDB**. 

### Запуск приложения
Для правильной работы приложения потребуется запустить сервер Node.js, работающий на 5000 порту, из папки **server/** командой ```npm run dev``` затем потребуется запустить само приложение из папки **tensor-js-app** командой ```npm start```.

### Возможности программы и особенности
Программа может отображать список студентов, полученный из облачной бд, и выполнять с ними операции добавления и удаления. 
Макет страницы поддерживает адаптивность.
